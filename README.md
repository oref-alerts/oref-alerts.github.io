# שאגת הארי — Pikud HaOref Alert History

Visual dashboard for IDF Home Front Command alert history across 1,700+ Israeli areas. Time-grid calendar with hours × days, color-coded by event type.

**[Live Dashboard](https://oref-alerts.github.io/)**

<!-- TODO: Add screenshot -->

## Features

- Time-grid calendar (rockets 🔴, warnings 🟠, drones 🟣)
- 1,700+ areas with search and multi-select
- Dark/light theme, responsive mobile layout
- Date picker with offset controls and pagination
- Hourly auto-updates

## Disclaimer

This dashboard is provided for **informational and educational purposes only**. It is **not a real-time alerting system** and must not be used as a substitute for official Home Front Command notifications. Data is sourced from the Pikud HaOref public API on a best-effort basis — **no guarantee of accuracy, completeness, or timeliness** is made. The authors are not responsible for any decisions made based on information displayed here. Always follow official instructions from Pikud HaOref during emergencies.
